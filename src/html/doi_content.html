API Documentation

GET /api/download/{doi}
Download a paper using its DOI identifier 
curl "https://campus.monius.uk/api/download/10.1038/nature12373" \
  -H "Accept: application/pdf" \
  --output paper.pdf
GET /api/metadata/{doi}
Get metadata for a paper 
curl "https://campus.monius.uk/api/metadata/10.1038/nature12373" \
  -H "Accept: application/json"
